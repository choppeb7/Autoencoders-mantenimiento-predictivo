-----------------------------------------------------
PARTE 0 — OBJETIVO DEL PROYECTO
-----------------------------------------------------

Objetivo general:
Construir un modelo de autoencoder que aprenda el comportamiento normal de un sistema industrial y detecte anomalías utilizando el error de reconstrucción.

Idea clave:
El modelo:
- aprende solo datos normales
- reconstruye inputs
- falla al reconstruir datos anómalos

Error alto = posible anomalía

Aplicación industrial:
- motores eléctricos
- bombas
- reductores
- sistemas térmicos
- instrumentación industrial

1. Definir dimensión de entrada

PARTE 7 — ERROR DE RECONSTRUCCIÓN
-----------------------------------------------------

Objetivo:
Convertir el modelo en detector de anomalías.

Pasos:
1. Predecir reconstrucción:
   X → X_hat
2. Calcular error:

3. Obtener:

   - error en anómalos

Concepto clave:
Error = anomaly score
